# Threads of Africa - Website Documentation

## Project Overview
**Threads of Africa** is a premium African streetwear fashion brand website that blends contemporary urban style with rich cultural heritage. The website showcases high-fashion South African streetwear collections, tells the brand story, and provides multiple ways for customers to connect with the brand.

## Brand Identity
- **Founded**: 2015 in Johannesburg by Lwazi Dlamini
- **Mission**: Transform streetwear into wearable art that connects wearers to South Africa's heritage
- **Vision**: Africa's premier high-fashion streetwear label
- **Values**: Creativity, Cultural Authenticity, Luxury Quality, Global Engagement
